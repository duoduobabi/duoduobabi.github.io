repo命令

1 repo init

repo init -u URL 在当前目录安装仓库，会在当前目录创建一个.repo目录。
-u 参数指定一个URL，从这个URL中取得仓库的manifest文件
-b 指定分支

2 repo sync

同步代码
-l only update working tree, don’t fetch
-c fetch only current branch from server

3 repo start

创建一个分支

4 repo abandon

删除一个分支

5 repo status

显示project中每个仓库状态

6 repo checkout

切换分支

7 repo info

project信息

8 repo list

project列表

9 repo branch(es)

查看所有分支

10 repo diff

查看修改

应用patch

vi ~/bin/repo-patch

chmod u+x ~/bin/repo-patch

repo-patch update.diff

ubuntu

mac

11 repo forall

-c 遍历所有的git仓库，并在每个仓库执行-c所指定的命令
-r 对指定仓库执行命令
